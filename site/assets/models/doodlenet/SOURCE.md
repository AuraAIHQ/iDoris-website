# DoodleNet model (self-hosted)

TensorFlow.js model that classifies hand-drawn doodles into 345 categories,
trained on Google's "Quick, Draw!" dataset.

- Source: https://github.com/yining1023/doodleNet (MIT License)
- Files: model.json + group1-shard1of1.bin (~2MB) + class_names.txt (345 labels)
- Rehosted here so idoris.ai serves it directly — no dependency on third-party hosting.

Used by /play/quick-draw-guess.
